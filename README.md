# Student_Acadamic_Reaport
## Project Overview
This Power BI project focuses on analyzing student academic performance to gain meaningful insights into marks distribution, subject-wise trends, and overall academic outcomes. The dashboard helps educators and decision-makers understand performance gaps, identify high-achieving students, and improve academic strategies.

## Objectives

- Analyze overall student performance
- Identify subject-wise strengths and weaknesses
- Visualize pass/fail trends
- Compare performance across classes or categories
- Support data-driven academic decisions

## Key Features

- Interactive Power BI Dashboard with dynamic visuals and slicers
- Overall Performance Analysis showing average marks and pass percentage
- Subject-wise Performance Insights to identify strong and weak subjects
- Pass vs Fail Comparison for quick academic evaluation
- Class-wise Performance Comparison to analyze variations across groups
- User-friendly Filters & Slicers for customized data exploration

## Key Insights

- Identifies subjects where students perform best and worst
- Highlights overall academic trends
- Helps detect performance gaps early
- Supports better academic planning and intervention

## Conclusion

This Power BI project successfully analyzes student academic performance and presents insights in a clear and interactive manner.
The dashboard helps identify performance trends, subject-wise strengths and weaknesses, and overall academic outcomes.
By using visual analytics and data-driven insights, this project supports better academic decision-making, early identification of performance gaps, and improved educational planning.
